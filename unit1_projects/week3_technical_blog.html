What are the great things about JavaScript? Why is it so popular?

JavaScript is popular among Web Developers mainly due to 4 great features:

1. Browser support

You don't have to install any plugin to access its content, as flash would for instance. All browsers have adopted JavaScript as a scripting language and they provide integrated support for it.

2. Can be used on client side as well as on server side

This characteristic by itself explain why there are so many JavaScript developers.

3. Functional programming language

In javascript, function could be assigned to variables just like any other data types. Not only that, but a function can accept another function as a parameter and can also return a function. You can have functions with no name as well. Clearly, this gives you the ability to code in functional programming style.

4. Support for objects

Javascript is an object oriented language. However, the way javascript handles objects and inheritance is bit different from conventional object oriented programming languages like Java. Due to this, javascript supports most of the object oriented concepts while being simple to learn and use.

.

What are it's limitations?

  • JavaScript cannot write to files on the server without the help of a server side script.
  • JavaScript cannot access databases unless you use Ajax and have a server side script perform the database accesses for you.
  • JavaScript cannot read from or write to files in the client.
  • JavaScript cannot close a window if it didn't open it.
  • JavaScript cannot access web pages hosted on another domain.
  • JavaScript cannot protect your page source or images.
